Bem Vindo ao K1Plaza!

Vamos configurar seu ambiente de desenvolvimento.

Para proseguir, é necessário que você possua uma conta no GitHub.

Já Possuo Criar minha conta

Como esse é um ambiente de desenvolvimento local, não é possível usar a forma tradicional (via OAuth) para conectarmos com sua conta do GitHub.

Para essas situações o GitHub fornece o recurso Personal access tokens, que permite que você crie um token OAuth manualmente para usar em scripts ou ferramentas como o K1Plaza.

1. Acessar seu painel de configurações no GitHub

Settings > Developer Settings > Personal Access Token

2. Criar novo token

Na página Personal Access Tokens, clique em para abrir o wizard.

Preencha com as seguintes informações:

Campo Valor
Token description K1Plaza Developer
Select scopes repo, user:read, user:email
Ver screenshot

3. Configurar o K1Plaza com seu token

Esse token será armazenado e utilizado apenas no seu ambiente local de desenvolvimento.

Nunca compartilhe esse token com ninguem. Você pode deletar ou recriar esse token a qualquer momento no GitHub.
